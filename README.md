# python-url-shortner(Flask)
A simple URL shortener web application built using Python and Flask.

## Features
- Generate short URLs for long links
- Redirect short URLs to the original website
- Simple HTML frontend
- In-memory URL storage

## Tech Stack
- Python
- Flask
- HTML

## How to Run
1. Install Flask
   pip install flask
2. Run the application
   python app.py
3. Open browser and go to
   http://127.0.0.1:5000
